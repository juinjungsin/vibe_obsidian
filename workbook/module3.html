Module 3 · Installation

따라해봅시다 — 설치

Obsidian 본체 설치 → Vault 생성 → 커뮤니티 플러그인 활성화 → Claudian 설치 → API 키 연결 → 동작 확인. 6단계.

소요 50분(실측 50.0분, 5분 단위) 방식 강사 시연 + 동시 실습 완료 시점 Claudian 호출 응답 1건
LEARNING GOALS
  1. 본인 노트북에 Obsidian + Claudian이 동작 가능한 상태로 설치된다.
  2. Vault 폴더 위치를 본인이 기억할 수 있는 경로에 둔다.
  3. Claudian에서 API 키가 정상 연결되어 첫 호출이 성공한다.
01 / STEPS

설치 6단계

총 35분
  1. Obsidian 다운로드 · 설치 5분
    obsidian.md 접속 → 본인 OS용 설치 파일 다운로드 → 실행 → 기본 옵션으로 설치 진행.
  2. Vault 생성 5분
    Obsidian 첫 실행 화면에서 Create new vault → 이름 vibe_2026 → 위치 선택. iCloud / OneDrive 동기화 폴더는 비권장.
  3. 커뮤니티 플러그인 활성화 5분
    Settings → Community plugins → Restricted mode OFF → Browse 가능 상태로 진입.
  4. Claudian 설치 8분
    Browse → 검색 Claudian → Install → Enable. 사이드바에 보라색 아이콘 노출 확인.
  5. API 키 연결 8분
    Claude 콘솔에서 API 키 발급 → Claudian 설정 패널 → API Key 입력 → Save → 상태 표시기 green 확인.
  6. 동작 확인 4분
    새 노트 작성 → Cmd/Ctrl + P → Claudian: Summarize → 응답 1건 수신.

04 / FOLDERS

최초 폴더 구조 만들기

실습

Vault 생성 직후 다음 5개 폴더를 만든다. 이름·번호는 강사 기본값을 따른다.

📁 vibe_2026/
├─ 00_inbox/      # raw 수집
├─ 10_atomic/     # 단일 개념 노트
├─ 20_wiki/       # 정제된 위키
├─ 30_publish/    # 외부 공유 후보
└─ 99_meta/       # 템플릿 · 스크립트
  • 좌측 파일 트리에서 우클릭 → New folder → 위 5개 이름 그대로 생성
  • 각 폴더 안에 .gitkeep 또는 빈 노트 readme.md 1개 추가 (선택)
  • Settings → Files & Links → Default location for new notes = 00_inbox 지정

모듈 3 종료 시 셀프 체크

  • Obsidian이 실행되고 Vault 이름이 화면 상단에 보인다.
  • 좌측 파일 트리에 5개 폴더가 만들어졌다.
  • 설정 → Community plugins에서 Claudian이 Enabled 상태다.
  • Claudian 설정 패널의 상태 표시기가 green이다.
  • Summarize 명령으로 응답 1건을 받았다.
