Module 6 · Troubleshooting

트러블슈팅

설치·인증·호출·동기화·자동화 5개 영역에서 자주 발생하는 오류를 분류하고, 본인 환경에서 재현되는 오류를 워크시트로 추적한다.

소요 30분 방식 오류 분류 + 본인 사례 추적 결과물 해결 기록 1건 + 미해결 1건 큐
LEARNING GOALS
  1. 오류를 5개 영역으로 분류해서 본다.
  2. 본인 환경 오류 1건 이상을 워크시트에 기록·해결한다.
  3. 해결 못한 오류는 미해결 큐로 옮겨 강사·동료에 공유한다.
01 / TAXONOMY

오류 5개 영역

분류 기준
① 설치 · 실행
Obsidian이 켜지지 않음. 플러그인이 목록에 없음.
② 인증 · API
API 키 거부. 401/403 응답. 결제 한도 초과.
③ 호출 · 응답
타임아웃. 응답 빈 문자열. 한국어가 영어로 나옴.
④ 파일 · 동기화
노트가 두 개로 쪼개짐. .conflict 파일 생성.
⑤ 자동화
트리거가 발화 안 됨. 동일 파일 무한 루프.
⑥ 기타
한글 파일명 깨짐. 그래프 뷰 멈춤.
02 / COMMON

자주 발생 오류 · 원인 · 해결

레퍼런스 표
영역증상 / 메시지원인해결
①Community plugins 메뉴 없음Restricted mode ONSettings → Community plugins → Turn off restricted mode
①Claudian 검색 결과 없음검색어 오타 / 사내망 차단정확히 Claudian 입력 / 외부 네트워크 확인
②401 unauthorizedAPI 키 만료 또는 오타키 재발급 → 설정 패널 재입력 → Save
②429 rate_limit분당 호출 한도 초과요청 간격 늘리기 · 플랜 업그레이드
②403 insufficient_credit결제 잔액 부족콘솔에서 결제 정보 확인
③응답 30초 후 타임아웃max tokens 과대 · 네트워크 지연tokens 1024로 낮춤 · 핫스팟 테스트
③응답이 영어로만 나옴Language 미설정설정 패널 Language → ko
③응답 빈 문자열입력 노트가 빈 본문본문 200자 이상 후 재시도
④*.conflict-2026-05-18.md 생성iCloud / OneDrive 동기화 지연로컬 폴더로 Vault 이전 / Obsidian Sync 사용
④노트 저장은 되는데 그래프 미반영인덱스 캐시 지연Cmd/Ctrl+P → Reload app
⑤자동화 트리거가 발화하지 않음경로 패턴 불일치 / debounce 미설정경로 점검 · 로그에서 매칭 여부 확인
⑤같은 파일 무한 루프 변환출력이 다시 입력 폴더에 저장됨출력 경로를 입력 패턴 밖으로 이동
⑥한글 파일명 깨짐 (macOS↔Win)유니코드 정규화 (NFD vs NFC)파일명에 영문 슬러그 사용 권장
⑥그래프 뷰가 5초+ 멈춤노드 1000+ · forces 과부하필터로 표시 노드 축소 / forces 기본값
