# etc2

트러블슈팅 보조 실습. 설치·인증·호출·동기화·자동화 5개 영역에서 자주 발생하는 오류를 분류하고, 본인 환경에서 재현되는 오류를 워크시트로 추적한다.

소요 **30분**(실측 28.1분, 5분 단위)
방식 **오류 분류 + 본인 사례 추적**
결과물 **해결 기록 1건 + 미해결 1건 큐**

**LEARNING GOALS**

1. 오류를 **5개 영역**으로 분류해서 본다.
2. 본인 환경 오류 **1건 이상**을 워크시트에 기록·해결한다.
3. 해결 못한 오류는 **미해결 큐**로 옮겨 강사·동료에 공유한다.

01 / TAXONOMY

## 오류 5개 영역

분류 기준

① 설치 · 실행

Obsidian이 켜지지 않음. 플러그인이 목록에 없음.

② 인증 · API

API 키 거부. 401/403 응답. 결제 한도 초과.

③ 호출 · 응답

타임아웃. 응답 빈 문자열. 한국어가 영어로 나옴.

④ 파일 · 동기화

노트가 두 개로 쪼개짐. .conflict 파일 생성.

⑤ 자동화

트리거가 발화 안 됨. 동일 파일 무한 루프.

⑥ 기타

한글 파일명 깨짐. 그래프 뷰 멈춤.

02 / COMMON

## 자주 발생 오류 · 원인 · 해결

레퍼런스 표

| 영역 | 증상 / 메시지 | 원인 | 해결 |
| --- | --- | --- | --- |
| **①** | Community plugins 메뉴 없음 | Restricted mode ON | Settings → Community plugins → Turn off restricted mode |
| **①** | Claudian 검색 결과 없음 | 검색어 오타 / 사내망 차단 | 정확히 `Claudian` 입력 / 외부 네트워크 확인 |
| **②** | `401 unauthorized` | API 키 만료 또는 오타 | 키 재발급 → 설정 패널 재입력 → Save |
| **②** | `429 rate_limit` | 분당 호출 한도 초과 | 요청 간격 늘리기 · 플랜 업그레이드 |
| **②** | `403 insufficient_credit` | 결제 잔액 부족 | 콘솔에서 결제 정보 확인 |
| **③** | 응답 30초 후 타임아웃 | max tokens 과대 · 네트워크 지연 | tokens 1024로 낮춤 · 핫스팟 테스트 |
| **③** | 응답이 영어로만 나옴 | Language 미설정 | 설정 패널 Language → `ko` |
| **③** | 응답 빈 문자열 | 입력 노트가 빈 본문 | 본문 200자 이상 후 재시도 |
| **④** | `*.conflict-2026-05-18.md` 생성 | iCloud / OneDrive 동기화 지연 | 로컬 폴더로 Vault 이전 / Obsidian Sync 사용 |
| **④** | 노트 저장은 되는데 그래프 미반영 | 인덱스 캐시 지연 | `Cmd/Ctrl+P` → `Reload app` |
| **⑤** | 자동화 트리거가 발화하지 않음 | 경로 패턴 불일치 / debounce 미설정 | 경로 점검 · 로그에서 매칭 여부 확인 |
| **⑤** | 같은 파일 무한 루프 변환 | 출력이 다시 입력 폴더에 저장됨 | 출력 경로를 입력 패턴 밖으로 이동 |
| **⑥** | 한글 파일명 깨짐 (macOS↔Win) | 유니코드 정규화 (NFD vs NFC) | 파일명에 영문 슬러그 사용 권장 |
| **⑥** | 그래프 뷰가 5초+ 멈춤 | 노드 1000+ · forces 과부하 | 필터로 표시 노드 축소 / forces 기본값 |

### etc2 종료 시 셀프 체크

- 오류 5개 영역의 **이름**을 말로 답할 수 있다.
- `claudian.log`에서 ERROR 라인 한 건을 **해석해 분류**했다.
- 본인 환경에서 만난 오류 **1건 이상**을 워크시트에 기록했다.
- 해결 못한 오류는 **미해결 큐**로 옮겼다.
