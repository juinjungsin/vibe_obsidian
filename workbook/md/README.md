# 워크북 모듈 수정용 Markdown

이 폴더는 `workbook/module1.html`부터 `workbook/module7.html`까지의 실습 내용을 수정 지시용 Markdown으로 옮긴 작업 원고입니다.

## 사용 방법

- 수정 요청은 `module1.md`처럼 파일명과 섹션 제목을 기준으로 전달하면 됩니다.
- 표의 빈칸은 `[작성]`, 긴 작성 영역은 `[작성칸]`, 자유 메모 영역은 `[메모 영역]`으로 표시했습니다.
- SVG로 만든 그래프/흐름도는 `[시각 자료]`로 요약했습니다. 시각 자료 문구를 바꾸려면 해당 표시를 기준으로 지시하면 됩니다.
- 이 Markdown은 배포 페이지가 아닙니다. 최종 수정은 원본 HTML에 반영한 뒤 로컬 확인과 GitHub 배포를 진행합니다.
