03 / HOW TO USE

워크북 사용 가이드

3 가지 모드
🖥️ 화면에서 보기
각 모듈을 브라우저에서 열고 메모는 외부 도구(예: Obsidian 본인 Vault)에 기록.

Cmd/Ctrl + P → PDF로 저장 또는 인쇄. 빈칸과 워크시트는 인쇄용 스타일 적용됨.
